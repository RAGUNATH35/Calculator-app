# Calculator App

A simple Calculator application built using Flutter and Dart.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear (C) button
- Responsive UI
- User-friendly design

## Technologies Used

- Flutter
- Dart

## Project Structure

```text
lib/
├── main.dart
```

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/RAGUNATH35/calculator-app.git
```

### Install Dependencies

```bash
flutter pub get
```

### Run the Application

```bash
flutter run
```
